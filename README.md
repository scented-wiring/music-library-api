# Music Library API

This API creates a database in which you can make HTTP requesrs to add and interact with artist, album and song data.

## Prerequisites

Requires express, mysql2 and sequelize as dependencies and chai, dotenv, mocha and supertest as dev-dependencies.

## Built With

- Docker
- MySQLWorkbench
